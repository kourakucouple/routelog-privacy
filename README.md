# 車旅きろく プライバシーポリシー

**最終更新日: 2026 年 5 月 8 日**

行楽夫婦(以下「当方」)が提供する Android アプリ「車旅きろく」(以下「本アプリ」)におけるユーザー情報の取り扱いについて、以下のとおり定めます。

---

## 2. 取得する情報

本アプリは、以下の情報を端末上で取得・利用します。

### (1) 位置情報
- GPS、Wi-Fi、モバイル基地局による現在地および移動軌跡
- バックグラウンドでの自動記録(ユーザーが「旅行モード」を有効にした期間に限る)

### (2) 写真・動画のメタデータ
- ユーザーが許可した場合のみ、端末内の写真・動画ファイルの URI、撮影日時、撮影位置情報(EXIF)を読み取り、旅の記録に紐付けます
- 写真・動画ファイルそのものを本アプリがコピー・保存することはありません

### (3) ユーザー設定情報
- 自宅として登録した地点の座標およびラベル
- 表示・通知・プライバシーに関する各種設定

---

## 3. 利用目的

取得した情報は、次の目的のためにのみ利用します。

- 旅行ごとの軌跡・滞在地・走行距離の記録と表示
- 自宅地点を起点とした自動記録の開始・停止
- 写真・動画と旅行記録との紐付け表示
- ユーザーの希望に応じた記録の書き出し(エクスポート)

---

## 4. 情報の保管とセキュリティ

- すべてのデータは **ユーザーの端末内のみ** に保存されます
- 当方が運営する **サーバーや第三者サーバーへの送信は一切行いません**
- データベースは **SQLCipher による暗号化**(AES-256)で保護されています
- スクリーンショット・画面録画は既定で抑止しています(設定で許可した場合のみ解除)
- 監査ログには操作の種別のみ記録し、座標や個人情報そのものは記録しません

ただし、地名表示のために国土地理院の住所変換 API へ座標を送信する場合があります。送信先・送信内容はアプリ内の設定で精度を調整できます。

---

## 5. 第三者提供

当方は、ユーザーの個人情報を第三者に提供することはありません。法令に基づく開示要請を除きます。

---

## 6. 広告・解析ツール

本アプリは、広告配信ネットワークおよびユーザー行動解析ツール(Google Analytics 等)を **一切使用していません**。

---

## 7. ユーザーの権利

ユーザーはいつでも以下を行うことができます。

- 設定画面の「全データを削除」(または準備中の旨の案内)から記録の削除
- 端末から本アプリをアンインストールすることで、すべてのデータを完全に削除
- 「バックアップを作成」で暗号化バックアップを取得し、別端末へ移行

---

## 8. 子どもの個人情報

本アプリは 13 歳未満の利用を想定していません。13 歳未満の方が利用する場合は、保護者の同意のもとで利用してください。

---

## 9. 権限の利用について

| 権限 | 利用目的 |
|------|------|
| 位置情報(精密) | 旅行軌跡の記録 |
| バックグラウンド位置情報 | 自動的な旅行記録の継続 |
| 通知 | 記録中の状態通知 |
| 写真・動画の読み取り | 旅と写真の紐付け(任意) |
| 端末活動認識 | 停止検出による省電力化 |
| バッテリー最適化からの除外 | 長時間記録の継続 |

各権限はユーザーが Android の設定画面でいつでも取り消すことができます。
